Scripter le déploiement d'un site Jekyll sur OVH avec lftp

Ubuntu 

						$ sudo apt-get update
						$ sudo apt-get install lftp
Mac OS 

						$ brew install lftp
					

							#!/bin/sh

							set -e

							. ./.env

							local_path=/home/user/repo/_site

							jekyll build

							lftp ftp://$user:$pwd@$host -e "
							  ls -R ;
							  mirror --reverse $local_path /www \
							         --delete ;
							  quit"
